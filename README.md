# SummarAIze | Summarize Articles using OpenAI

### About

An article summarizer which uses an OpenAI API to summarize text provided via fetched websites.

### Built With

- React
- Vite
- Redux / Redux Toolkit
- OpenAI API via RapidAPI
- TailwindCSS
- ShadCN UI
